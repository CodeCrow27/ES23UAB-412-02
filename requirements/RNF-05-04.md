# Requirement RNF-05

### ID:
RNF-05

### Type/Category:
No Funcional

### Version:
1.0

### Title:
Forma de pago
### Description:
Para realizar el pago mediante tarjeta bancaria o PayPal, el sistema deberá conectarse con la plataforma externa correspondiente para realizar la autenticación de datos y confirmar la transacción económica.
### Relations:
RF-22

### Comments:
Requisito no funcional de tipo Interficie externa
