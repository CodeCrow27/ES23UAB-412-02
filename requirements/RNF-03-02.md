# Requirements RNF-03

### ID:
RNF-03

### Type/Category:

### Version:
1.1

### Title:
Orden de platos
### Description:
El sistema debe mostrar primero los platos y menus que cumplan con las restricciones alimenatarias y ordenar las disponibles en funcion de las distancia entre el cliente y el cocinero.
